### Docs

- This project uses the following dependencies:
   - `django 4`
   - `python 3.11` (should work for 3.9 and above)
   - `GNU Make` for running small scrips e.g tests

- For help, run:
    - `make help`
- To run the project:
    - `make run`
- To run tests:
    - `make test`
- For test coverage report:
    - `make coverage` (Currently above 90%)
